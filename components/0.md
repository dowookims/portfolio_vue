# MovieSlider
<br />
Honey BEE에서 제가 핵심이라고 생각하는 부분은 MovieSlider입니다. 특정 주제에 맞는 영화 리스트를 보여주고, 사용자의 클릭, 마우스 이동에 따라 보여지는 UI를 변화 시키는 것 입니다.
저는 이것을 구현하기 위해 MovieSlider를 컴포넌트화 하였습니다.
